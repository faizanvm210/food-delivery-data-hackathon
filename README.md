# food-delivery-data-hackathon

## Objective
Merge transactional, user, and restaurant data to analyze order trends, user behavior,
city-wise revenue, and membership impact.

## Files Used
- orders.csv
- users.json
- restaurants.sql

## Key Operations
- Data loading using Pandas and SQLite
- LEFT JOIN on user_id and restaurant_id
- Feature analysis for revenue, cuisine, ratings, and membership

## Output
- final_food_delivery_dataset.csv

## Tools
- Python
- Pandas
- SQLite
- Jupyter Notebook
